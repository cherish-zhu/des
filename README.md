技术实现
以PHP开发语言为基础的，基于ThinkPHP开发框架的开发的接近与内容开发框架的内容管理系统，采用MySQL作为主要数据库，MVC模式，全面开源。建立于RBAC的权限控制基础，支持Win/Linux系统环境下部署，最佳WEb服务器Nginx/Apache。
